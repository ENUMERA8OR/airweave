Airweave provides a Python and TypeScript SDK to help you integrate with your applications and extend functionality.


## Python SDK

```bash
pip install airweave-sdk
```

The Python SDK provides a complete interface to Airweave's functionality:

```python
from airweave import AirweaveClient

client = AirweaveClient(api_key="your-api-key")

# List all sources
sources = client.sources.list()

# List all source connections
connections = client.sources.list_connected_integrations(
  integration_type="source"
)

# Create one-off sync job
source_connection_id = connections[0].id

job = client.sync.create_sync(
  name="My first sync",
  source_connection_id=source_connection_id,
  run_immediately=True
)
```

## TypeScript/JavaScript SDK

```bash
npm install @airweave/sdk
# or
yarn add @airweave/sdk
```

The TypeScript SDK provides type-safe access to Airweave:

```typescript
import { AirweaveClient } from "@airweave/sdk";

const client = new AirweaveClient({
  apiKey: "your-api-key",
});

// List all sources
const sources = await client.sources.list();

// List all source connections
const connections = await client.sources.list_connected_integrations({
  integration_type: "source",
});

// Create one-off sync job
const job = await client.sync.create_sync({
  name: "My first sync",
  source_connection_id: connections[0].id,
  run_immediately: true,
});
```


## SDK Features

All SDKs provide:

- **Type Safety**: Full TypeScript/Python type definitions
- **Authentication**: API key and OAuth2 support
- **Error Handling**: Detailed error messages and retry logic
- **Rate Limiting**: Automatic rate limit handling
- **Async Support**: Native async/await patterns
